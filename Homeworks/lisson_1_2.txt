Ресурс для виконання ДЗ буде https://api.adviceslip.com/
Це сайт, який має API для отримання життєвих порад.
Тепер завдання:
1) Написати функцію, яка отримує рандомну пораду і повертає її у вигляді строки.
2) Написати функцію, яка отримує пораду за її ID (ціле число) і повертає її у вигляді строки.
3) Написати функцію, яка використовує першу функцію, щоб отримати задану кількість унікальних порад.
Кількість порад має передаватись в функцію, як параметр advice_count.
4) Написати функцію, яка використовує другу функцію, щоб отримати задану кількість унікальних порад.
Кількість порад має передаватись в функцію, як параметр advice_count.
Для формування ID можна використовувати range(advice_count)
